# 🐍 NoteCaddy

Помощник для заметок с интеллектуальной обработкой текста. Работает в двух режимах: CLI и веб.

## Установка

```bash
# Клонируйте репозиторий или скопируйте notecaddy.py

# Установите зависимости (для веб-режима):
pip install flask

# Опционально для красивого CLI:
pip install rich
```

## Быстрый старт

```bash
# Запуск веб-сервера (по умолчанию)
python notecaddy.py

# Или с указанием порта
python notecaddy.py --port 8080
```

## Режимы работы

### Веб-интерфейс

```bash
python notecaddy.py
# или явно
python notecaddy.py --web
```

Откроется в браузере: `http://localhost:5000`

**Возможности веб-интерфейса:**
- Форма ввода текста заметки
- Просмотр всех заметок (карточки)
- Отображение обработанного и исходного текста
- Удаление заметок

### CLI

```bash
# Создать заметку
python notecaddy.py add "Купить продукты молоко хлеб яйца. Завтра встреча с Анной в 15:00."

# Показать все заметки
python notecaddy.py list

# Удалить заметку
python notecaddy.py delete 1

# Принудительный CLI-режим
python notecaddy.py --cli list
```

## Возможности

### Интеллектуальная обработка текста

Приложение автоматически:

1. **Выделяет даты и время** — события с датами/временем выносятся в отдельный блок
2. **Определяет списки** — слова через запятую/точку с запятой преобразуются в списки
3. **Генерирует краткое резюме** — первое предложение или первое событие

#### Пример

Входной текст:
```
Купить продукты молоко хлеб яйца. Завтра встреча с Анной в 15:00, обсудить проект. Позвонить врачу.
```

Результат:
```
📅 События и даты:
  • Завтра встреча с Анной в 15:00, обсудить проект.

📋 Списки:
  • Купить продукты
  • молоко
  • хлеб
  • яйца

📝 Заметки:
  • Позвонить врачу.
```

## Структура JSON

Заметки сохраняются в файл `notes.json`:

```json
[
  {
    "id": 1,
    "created_at": "2026-03-02 11:09:27",
    "raw_text": "Купить продукты...",
    "processed_text": "📅 События...\n📋 Списки...",
    "summary": "Завтра встреча с Анной в 15:00..."
  }
]
```

## Зависимости

|-------|------------|-------------|

## Архитектура

```
notecaddy.py
├── Константы         # NOTES_FILE, DEFAULT_PORT
├── Ядро (Core)       # load_notes, save_notes, add_note, delete_note
├── Обработка текста  # process_text() - интеллектуальное форматирование
├── CLI               # run_cli_add, run_cli_list, run_cli_delete
├── Web (Flask)       # create_flask_app, HTML-шаблон
└── Точка входа       # main() - определение режима
```
